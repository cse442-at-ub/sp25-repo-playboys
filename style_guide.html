Music Web App UI Style Guide

Home Page

Colors

Primary: #1DB954

Background: #121212

Text: #ffffff

Fonts

Use font-family: Arial, sans-serif;

Reusable UI Components

Button (with hover functionality)

Code:

<button class="button-example">Click Me</button> .button-example { background-color: #ffffff; color: #000000; padding: 10px 20px; border: none; border-radius: 5px; cursor: pointer; transition: background-color 0.3s; } .button-example:hover { background-color: #00ff00; }

Input Fields

Code:

<input type="text" class="input-example" placeholder="Type here...">

Cards

Card Title

This is an example of a reusable card component.

Code:

<div class="card"> <h3>Card Title</h3> <p>This is an example of a reusable card component.</p> </div>

Navigation Structure

Use the following structure for linking pages:

<a href="profile.html">Go to Profile</a>

Consistent Layout

All pages should include a container with max-width 800px for consistency.

<div class="container"> Page content here... </div>
